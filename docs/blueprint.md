# **App Name**: Chromatic Grid

## Core Features:

- CMYK Input: Input fields for CMYK values (C, M, Y, K - 0 to 100).
- Grid & Variation Config: Selection fields for grid size (e.g., 15x15) and variation step (e.g., ±5%).
- CMYK Grid Generation: Generates a grid of CMYK color variations around the input color, labeling each block with its CMYK values.
- Color Preview: Display the CMYK colors correctly on the screen, with appropriate CMYK to RGB conversion.
- Scalable Printing: Printable output for larger formats (not just A4).
- Export Options: Options to download the grid as a PDF file and also open a print-friendly version for easy printing.

## Style Guidelines:

- Primary color: A moderate blue (#6495ED) for a calming yet technical feel.
- Background color: A light gray (#F0F0F0) that provides a neutral backdrop.
- Accent color: A contrasting orange (#FFA07A) to highlight interactive elements and important information.
- Font: 'Inter' sans-serif for all UI elements for its clean and legible style, suitable for both headlines and body text.
- A clear, modular grid layout ensuring easy readability of color values and grid blocks.
