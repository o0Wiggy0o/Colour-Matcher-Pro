**What is Colour Matcher Pro?**

Colour Matcher Pro is designed specifically for print designers, sign makers, prepress operators, and other color professionals. It aims to bridge the gap between on-screen color exploration and production-ready print work by providing a suite of powerful tools for generating, managing, and exporting CMYK color data. It runs entirely in your browser—no installation needed.

Here’s a breakdown of the core features:

---

### **Key Features**

**1. The Grid Generator**
This is the heart of the app. It's designed for creating and exploring vast palettes of color variations.

*   **Free Features:**
    *   **Manual Axis Mode:** Define a base CMYK color and generate an interactive grid by varying two CMYK values (e.g., Cyan and Magenta) across the X and Y axes.
    *   **Interactive Grid:** Hover over any color swatch to see its CMYK, RGB, and HEX values in real-time.
    *   **Color History:** Click a swatch to save it to your history for later use.
    *   **Export History:** Export your saved colors to CSV, JSON, or TXT.
    *   **Print Preview:** Generate a simple, printer-friendly HTML view of your grid with CMYK values overlaid.

*   **Pro Features:**
    *   **AI Variations Mode:** Provide a base color and let the AI generate a grid of unique, subtle variations by adjusting all four CMYK values simultaneously.
    *   **CMPTone® Color Matching:** Find colors from an extensive CMPTone® library and instantly get its closest CMYK equivalent to use as your base color.
    *   **Download CMYK PDF:** Generate a true CMYK PDF of your grid, ready for professional RIP software and printing environments.
    *   **CMPTone® Swatch Book Printing:** Create a multi-page, visually-sorted swatch book PDF with your own production notes (printer, media, profile) for a perfect physical reference.

**2. The Image Color Extractor**
A powerful tool for generating palettes and gradients directly from images and vector files.

*   **Free Features:**
    *   **Raster Image Color Picking:** Upload a JPG or PNG, hover to see a real-time "loupe" with CMYK values, and click to add colors to a printable test strip.

*   **Pro Features:**
    *   **SVG Palette Extraction:** Upload an SVG file (like a client's logo) and the app will parse it and extract every unique color into a palette.
    *   **AI Gradient Detection:** The AI analyzes your image to find the most prominent color gradients, which you can then print as a continuous-tone CMYK test strip.
    *   **CMPTone® Matching:** The print strip PDF will include the closest CMPTone® match for each color picked.

**3. The Colour Tracker (Pro Only)**
A project management tool to track color evolution and maintain consistency across jobs.

*   **Job Management:** Create different projects (e.g., "Client X Logo").
*   **Color History per Job:** Each job has its own persistent color history with notes, printer used, and standard rendering profiles (GRACol, FOGRA).
*   **Similar Color Search:** Select a color and instantly find visually similar colors across *all* jobs to find visually similar colors.
*   **AI-Powered Suggestions:** When you update a color in one job, the AI can suggest proportional updates for similar colors in other jobs.
